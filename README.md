# Mac Excel Shortcuts

Run Windows Excel shortcuts on Mac, and get a native Windows experience on Mac. Get your work done fast.

Mac Excel Shortcuts is a macOS app that maps the Excel keyboard shortcuts you already know from Windows onto macOS, so switching between platforms doesn't mean relearning your muscle memory. Shortcuts run entirely on-device — nothing is sent to the cloud, and no internet connection is required.

## Installation

### 1) Download the .dmg

Download the app from [maccove.com](https://maccove.com).

Open the downloaded `.dmg` and drag Mac Excel Shortcuts into your `Applications` folder. Installation takes seconds and requires zero setup.

### 2) Add permissions

Mac Excel Shortcuts needs two macOS permissions to work:

- **Accessibility** — to access Excel.
- **Input Monitoring** — to connect your keyboard to Excel.

Grant both under **System Settings → Privacy & Security**, then enable Mac Excel Shortcuts in the Accessibility and Input Monitoring lists.

### 3) Open Excel and start using shortcuts

Open Excel and start using your familiar Windows Excel shortcuts — no further configuration needed.
